#  Fraud Case Files

> *"Understanding fraud is the first step to stopping it."*

A collection of in-depth analyses of real financial fraud cases, examined through two complementary lenses: **behavioral psychology** and **technical indicators**.

Each case file follows a structured methodology inspired by threat intelligence frameworks (MITRE ATT&CK, ENISA), enriched with a behavioral dimension — because fraud is, above all, a human problem.

##  Methodology

Each case file is structured around 7 sections:

1. **Case Summary** — What happened and why it matters
2. **Attack Flow** — Step-by-step breakdown of the fraud
3. **Psychological Techniques** — Human biases and manipulation tactics exploited
4. **Technical Indicators** — Red flags detectable by systems
5. **Detection Opportunities** — What could have caught it earlier
6. **Prevention** — What should have been in place
7. **Lessons Learned** — Key takeaways for fraud analysts
